HTML básico - códigos HTML

Aprenda os princípios básicos para confecção de um site em HTML para que você possa fazer o seu primeiro site.

Motivação

HTML (abreviação de Hyperlink Markup Language) é uma linguagem de marcação utilizada na estruturação de páginas web. Sua sintaxe é bastante simples e, assim como XLM é baseada em tags, que representam os diversos elementos de uma página, como imagens e links.
